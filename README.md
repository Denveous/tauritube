# Tauritube

A small desktop YouTube client built with Tauri.

Search YouTube or paste a video, playlist, channel, or Shorts URL. Tauritube keeps local history, subscriptions, queue items, and blocks without requiring a YouTube login.

## Run

```powershell
npm install
npm run app
```

## Build

```powershell
npm run build
```

Installers are written to `src-tauri\target\release\bundle`.
